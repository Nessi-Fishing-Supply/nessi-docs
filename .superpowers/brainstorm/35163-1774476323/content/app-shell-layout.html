Three-column: Sidebar + Canvas + Detail Panel

Matches the playground exactly. Sidebar has main nav + contextual sub-nav. Canvas is the main visualization area (pan/zoom SVG). Detail panel shows contextual info for whatever's selected. Works for all 5 page types.

Layout Details

  • Topbar (48px): Logo + site URL. Minimal — no tabs (those moved to sidebar).
  • Sidebar (220px): 5 page links at top. Contextual sub-nav appears below separator when on Journeys or Lifecycles pages. Scrollable.
  • Canvas (flex): SVG with pan/zoom for Journeys and Lifecycles. Scrollable list for API Map, Data Model, Coverage.
  • Detail Panel (280px): Always visible. Shows contextual details for the selected item on any page. Empty state when nothing selected.

This is the layout we already have in the playground. The question is just: does this translate directly to the Next.js app, or do you want any structural changes?
